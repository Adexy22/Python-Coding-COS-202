## Project Description

The Personal Pocket CGPA Calculator (PPC) is a Python program designed to help students calculate their CGPA quickly and accurately. It accepts course units and letter grades, computes the total grade points and total units, and displays the final CGPA.

## Features

- Accepts multiple courses.
- Accepts grades A, B, C, D, E, and F.
- Calculates total course units.
- Calculates total grade points.
- Computes and displays the CGPA.
- Validates invalid grade entries.

## Grading System

| Grade | Grade Point |
|-------|------------:|
| A | 5 |
| B | 4 |
| C | 3 |
| D | 2 |
| E | 1 |
| F | 0 |

## How to Run

1. Install Python 3.
2. Open the project folder.
3. Run:

```bash
python cgpa.py
```

## Output

The program displays:

- Total Course Units
- Total Grade Points
- Final CGPA
